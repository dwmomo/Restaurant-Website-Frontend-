🍽️ Restaurant Website (Beginner Frontend Project)
This is my beginner-friendly restaurant website project built using only HTML and CSS.
⚠️ Note: This is a non-responsive website and is currently optimized for desktop view only.

🚀 About The Project
This project helped me understand:
->Structuring a webpage using semantic HTML
->Creating layouts using CSS Flexbox
->Adding smooth effects using CSS Animations
->Styling a clean and modern restaurant UI
->Organizing frontend project files properly
->It is a static frontend project — no JavaScript used.

🛠️ Built With
HTML5
CSS3
Flexbox
CSS Animations

✨ Features
🍴 Elegant restaurant homepage
📋 Navigation bar
🥗 Menu section with dish cards
🖼️ Image styling & positioning
🎨 Hover effects
✨ Smooth animations
📌 Clean and structured layout

📂 Project Structure
Restaurant-Website/
│
├── index.html
├── style.css
└── images/

📚 What I Learned
How Flexbox works (justify-content, align-items, flex-direction)
How to align multiple divs in the same row
How to create hover effects
How to use keyframes for animations
How to design a basic restaurant UI

⚠️ Limitations
❌ Not responsive
❌ No media queries used
❌ Desktop view only
❌ No JavaScript functionality

💡 Why I Built This
As a beginner in frontend development, I wanted to practice building a real-world layout without using frameworks.
This project helped me build confidence in CSS styling and layout design.
