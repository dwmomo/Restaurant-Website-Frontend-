# 🍽️ Restaurant Website (Beginner Frontend Project)

This is a beginner-friendly restaurant website built using only **HTML5 and CSS3**.

⚠️ **Note:** This project is completely non-responsive and optimized for desktop view only.

---

## 🚀 About The Project

This project was created to practice frontend fundamentals and understand how layouts work using pure CSS.

It is a static frontend website with no JavaScript or frameworks used.

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Flexbox  
- CSS Animations  

---

## ✨ Features

- 🍴 Elegant restaurant homepage  
- 📋 Navigation bar  
- 🥗 Menu section with dish cards  
- 🖼️ Image styling and positioning  
- 🎨 Hover effects  
- ✨ Smooth CSS animations  
- 📌 Clean and structured layout  

---

## 📂 Project Structure

Restaurant-Website/
│
├── index.html

└── images/

---

## 📚 What I Learned

- How Flexbox works (justify-content, align-items, flex-direction)  
- How to align multiple divs in the same row  
- How to create hover effects  
- How to use keyframes for animations  
- How to design a basic restaurant UI  

---

## ⚠️ Limitations

- ❌ Not responsive  
- ❌ No media queries used  
- ❌ Desktop view only  
- ❌ No JavaScript functionality  

---

## 💡 Why I Built This

As a beginner in frontend development, I wanted to practice building a real-world layout without using frameworks.

This project helped me build confidence in CSS styling and layout design.
